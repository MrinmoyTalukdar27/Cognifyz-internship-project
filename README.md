### Cognifyz Data Analysis Internship Project

## Overview
This repository contains the tasks completed during my Data Analysis Internship at Cognifyz Technologies. The project focuses on analyzing restaurant data using Python, Pandas, and Matplotlib to extract meaningful insights and create visualizations.

---

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Jupyter Notebook

---

## Dataset
The dataset contains restaurant-related information such as:
- Restaurant names
- Cities
- Cuisines
- Ratings
- Price ranges
- Online delivery availability
- Votes and popularity

---

# Level 1 Tasks

## 1. Top Cuisines Analysis
- Identified the top 3 most common cuisines
- Calculated percentage distribution
- Created bar chart and pie chart visualizations

## 2. City Analysis
- Identified the city with the highest number of restaurants
- Calculated average ratings by city
- Determined the city with the highest average rating

## 3. Price Range Distribution
- Analyzed restaurant price categories
- Visualized distribution using bar charts
- Calculated percentage distribution

## 4. Online Delivery Analysis
- Determined percentage of restaurants offering online delivery
- Compared ratings between restaurants with and without online delivery

---

# Level 2 Tasks

## 1. Restaurant Ratings Analysis
- Analyzed aggregate rating distribution
- Identified the most common rating ranges
- Calculated average number of votes

## 2. Cuisine Combination Analysis
- Identified the most common cuisine combinations
- Analyzed cuisine combinations with the highest ratings

## 3. Geographic Analysis
- Visualized restaurant locations using scatter plots
- Identified restaurant clustering patterns

## 4. Restaurant Chain Analysis
- Identified restaurant chains in the dataset
- Analyzed chain popularity and ratings

## Key Insights
- North Indian cuisine is the most common cuisine in the dataset.
- New Delhi has the highest number of restaurants.
- Restaurants with online delivery generally have higher ratings.
- Several restaurant chains show strong popularity based on votes.
